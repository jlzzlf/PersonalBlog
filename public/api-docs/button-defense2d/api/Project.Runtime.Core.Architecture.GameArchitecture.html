Remarks

该方法会在 JLZFramework.Architecture<T>.Init() 中且仅在架构首次初始化时调用一次。

通常应在这里完成 System、Model、Utility 的注册工作，例如： 

RegisterModel(new PlayerModel());
RegisterSystem(new CombatSystem());
RegisterUtility(new InputService());

不建议在此方法中直接执行业务逻辑。 它的职责应聚焦在“组装架构对象”而不是“运行业务流程”。
