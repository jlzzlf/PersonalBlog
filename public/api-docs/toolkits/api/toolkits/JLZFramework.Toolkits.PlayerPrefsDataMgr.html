Remarks

该工具会把数据对象的 public 字段拆分后存入 PlayerPrefs，读取时根据类型和 key 重新创建并组装对象。 使用约束：存档对象请作为纯数据结构使用，只声明需要持久化的 public 实例字段，不支持属性、private 字段和静态字段。 字段类型支持 string、int、float、bool、List<T>/Dictionary<TKey,TValue> 这类可实例化集合，以及由这些类型组成的自定义数据类型。 自定义数据类型和集合类型需要能通过无参构造函数创建，字段值不要为 null。 存档 key 依赖数据类型名、字段类型名和字段名，重命名类型或字段会导致旧存档无法按原规则读取。
