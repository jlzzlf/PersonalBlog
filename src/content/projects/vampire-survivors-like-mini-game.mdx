## 项目简介

这个项目参考了《吸血鬼幸存者》的核心体验，目标是先把“移动闪避 + 自动攻击 + 敌潮压迫感”的玩法循环跑顺，再逐步补上成长与反馈层。

当前版本先把刷怪、命中、回收与战斗节奏搭成一条完整链路，让原型具备一个可以持续迭代的基础骨架。

## 当前重点

原型阶段最需要优先解决的是高频刷怪场景下的稳定性。敌人、投射物和掉落物会被不断创建，如果结构不提前收好，后面越加功能越难维护。

- 使用对象池复用敌人、投射物和掉落物，降低频繁 `Instantiate / Destroy` 的开销。
- 围绕持续刷怪和自动攻击搭建基础战斗主循环，让原型先具备“能玩、能调、能扩”的结构。
- 为后续加入技能升级、更多敌人行为和 UI 反馈预留清晰扩展空间。

## 下一步计划

- 继续补充成长系统，让角色在局内拥有更明确的升级与构筑感。
- 增加敌人类型和刷怪节奏变化，让战斗不只是数量堆叠。
- 完善受击、掉落、结算等反馈层，让玩法体验更完整。
