# 单位圆与三角函数

在单位圆中，一个角度 θ 对应圆上的一点：

<svg
  viewBox="0 0 300 240"
  width="100%"
  style={{ maxWidth: "420px", border: "1px solid var(--color-border)", borderRadius: "12px" }}
>

  <line x1="20" y1="120" x2="220" y2="120" stroke="black" strokeWidth="1" />
  <line x1="120" y1="220" x2="120" y2="20" stroke="black" strokeWidth="1" />

  {/* x / y 标记 */}

    fill="none"

  <circle cx="176.6" cy="63.4" r="4" fill="black" />

    x1="120"
    y1="120"
